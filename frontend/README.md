# ZEWD Frontend SPA

## Tech Stack
- Vue 3 (Vite)
- Tailwind CSS
- Pinia (State)
- Vue Router
- Axios

## Setup
1. Install dependencies: `npm install`.
2. Start dev server: `npm run dev`.

## Environment
Ensure the backend API is running at `http://localhost:8000` (configurable in `src/services/api.js`).
